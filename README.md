# irnm - improved rofi-network-manager
## rofi interface utility for NetworkManager

### Dependencies

```
sudo pacman -S bash
yay -S rofi dunstify
```

### Installation

```
git clone https://github.com/cherrynoize/irnm irnm
cd irnm
chmod +x install.sh
./install.sh
```

### Usage

```
irnm
```
